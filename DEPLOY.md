# 배포 안내

## 1. Vercel 연결

1. Vercel에 로그인합니다.
2. `Add New` → `Project`를 누릅니다.
3. GitHub 저장소 `shupapa05/sail-digital-citizenship`을 선택합니다.
4. 아래처럼 설정합니다.

```text
Framework Preset: Other
Build Command: 비워둠
Install Command: 비워둠
Output Directory: .
```

5. `Deploy`를 누릅니다.

## 2. 주소

배포 후 주소는 보통 아래처럼 만들어집니다.

```text
https://sail-digital-citizenship.vercel.app/
```

교사용 주소는 다음과 같습니다.

```text

```
